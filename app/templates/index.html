üì± B.3: Remote Smart Charging

This simulates smart charging where the EV is already plugged in and the system decides when to start charging.
The Central System sends a RemoteStartTransaction command (no physical RFID card tap needed).

Use cases:

  • Solar-optimized charging (EVCC, SolarManager) - start when solar surplus available
  • Dynamic pricing optimization - charge during off-peak hours
  • Grid load balancing - start/stop based on grid capacity
  • App/web payment - user pays via app or credit card, system starts charging remotely
  • QR code payment - scan QR, pay, charging starts automatically
